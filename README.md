# FloralDET

A mobile application to detect following 17 categories of flowers.

Alpine sea holly,
Balloon flower,
Bluebell,
Buttercup,
Cowslip,
Crane Flower,
Crocus,
Daffodil,
Daisy,
Dandelion,
Hibiscus,
Pansy,
Passion,
Rose,
Snowdrop,
Sunflower,
Tigerlily.
